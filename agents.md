# Урок: создаём игру «Змейка» в pygame

Пошаговый урок для детей 12–13 лет. Каждый шаг — отдельный файл. Запускайте его и смотрите, что получилось, прежде чем переходить дальше.

**Установка pygame** (один раз в терминале):

```
py -m pip install pygame
```

**Как запускать каждый шаг:**

```
py snake-1.py
py snake-2.py
...
```

---

## Шаг 1. Полотно (окно игры)

**Файл:** `snake-1.py`

**Что делаем:** создаём чёрное окно — это «полотно», на котором будет игра.

**Главные идеи:**
- `pygame.init()` — включаем библиотеку
- `display.set_mode()` — создаём окно нужного размера
- `while running` — главный цикл игры: пока окно открыто, программа работает
- `screen.fill()` — заливаем фон цветом
- `display.flip()` — показываем кадр на экране
- `clock.tick(60)` — ограничиваем скорость до 60 кадров в секунду

**Проверка:** должно открыться чёрное окно. Закрывается крестиком.

---

## Шаг 2. Голова змейки

**Файл:** `snake-2.py`

**Что добавляем:** зелёный квадрат — голова змейки.

**Главные идеи:**
- `CELL = 20` — размер одной «клетки» на поле (как в шахматах)
- `snake = [(300, 200)]` — список с координатами частей змейки; пока только голова
- `pygame.draw.rect()` — рисуем прямоугольник: экран, цвет, позиция и размер

**Проверка:** чёрное окно с одним зелёным квадратом по центру.

---

## Шаг 3. Управление с клавиатуры

**Файл:** `snake-3.py`

**Что добавляем:** змейка движется сама; стрелками меняем направление.

**Главные идеи:**
- `direction = (CELL, 0)` — вектор движения: вправо на 20 пикселей за шаг
- `pygame.time.set_timer()` — таймер: каждые 150 мс змейка делает шаг
- `event.type == pygame.KEYDOWN` — нажатие клавиши
- `K_UP`, `K_DOWN`, `K_LEFT`, `K_RIGHT` — стрелки
- нельзя развернуться на 180° (нельзя идти влево, если сейчас идёшь вправо)

**Проверка:** змейка ползёт вправо; стрелками можно поворачивать.

---

## Шаг 4. Появление яблок

**Файл:** `snake-4.py`

**Что добавляем:** красное яблоко в случайном месте на поле.

**Главные идеи:**
- `import random`
- `random.randint()` — случайное число для координат
- координаты кратны `CELL`, чтобы яблоко попадало ровно в клетку
- `spawn_apple()` — функция, которая ставит яблоко не на змейку

**Проверка:** змейка движется, на поле есть красный квадрат-яблоко.

---

## Шаг 5. Хвост и рост при съедании яблока

**Файл:** `snake-5.py`

**Что добавляем:** полная логика змейки.

**Главные идеи:**
- змейка — список сегментов: `[голова, сегмент, сегмент, ...]`
- каждый шаг: новая голова впереди, хвост убирается — змейка «ползёт»
- если голова попала на яблоко: хвост **не** убираем → змейка длиннее; появляется новое яблоко
- если голова ударилась о стену или о себя — игра заканчивается
- `font.render()` — текст «Игра окончена» на экране

**Проверка:** съедая яблоки, змейка растёт; при столкновении игра останавливается.
