## Урок 1. Класс и объект (~20 мин)

**Файл:** `lesson-1.py`  
**Тема:** что такое класс, объект, атрибуты

### Теория (5 мин)

- **Класс** — чертёж. `class Point` описывает «точку на плоскости».
- **Объект** — конкретная точка, созданная из чертежа: `a = Point(3, 5)`.
- **`__init__`** — вызывается при создании; задаёт начальные значения.
- **`self`** — сам объект. `self.x = x` сохраняет координату внутри точки.
- **Атрибут** — «поле» объекта: `a.x`, `a.y`.

### Практика (10 мин)

1. Запустите `lesson-1.py` — увидите две точки.
2. Создайте третью точку `c = Point(0, 0)` и выведите её координаты.
3. Измените `b.x = 10` и снова запустите — объект можно менять.

### Мини-задание (5 мин)

Создайте точку с вашими любимыми координатами и выведите: «Моя точка: (x, y)».

**Проверка:** понимаете разницу между классом `Point` и объектом `Point(1, 2)`.

## Урок 2. Методы (~20 мин)

**Файл:** `lesson-2.py`  
**Тема:** методы — что умеет делать объект

### Теория (5 мин)

- **Метод** — функция внутри класса. Первый аргумент всегда `self`.
- `point.move(2, -1)` — точка **сама** сдвигается.
- `point.info()` — точка **сама** рассказывает о себе.
- Методы группируют данные и действия в одном месте.

### Практика (10 мин)

1. Запустите `lesson-2.py`.
2. Вызовите `a.move(1, 1)` два раза подряд — куда попадёт точка?
3. Добавьте метод `reset()`, который возвращает точку в `(0, 0)`.

### Мини-задание (5 мин)

Создайте точку, сдвиньте её три раза методом `move`, выведите результат через `info()`.

**Проверка:** умеете вызывать методы и видите, как меняются `x` и `y`.

**Файл:** `lesson-6.py`

## Урок 7. `__repr__` (~20 мин)

**Файл:** `lesson-7.py`  
**Тема:** техническое описание для отладки

### Теория (5 мин)

- **`__repr__`** — «описание для программиста».
- `repr(point)` → `Point(3, 5)` — похоже на код для создания.
- В списках Python использует `__repr__`, не `__str__`.
- Можно иметь **оба**: `__str__` — коротко, `__repr__` — подробно.

### Практика (10 мин)

1. Запустите `lesson-7.py`, сравните `print(a)` и `repr(a)`.
2. Выведите список точек — увидите `__repr__` у каждой.
3. В `Lamp.__repr__` используйте `repr(self.position)`.

### Мини-задание (5 мин)

Напишите `__repr__` так, чтобы по строке было видно, включена лампа или нет.

**Проверка:** понимаете разницу между «для человека» (`__str__`) и «для кода» (`__repr__`).

## Урок 8. `__add__` (~20 мин)

**Файл:** `lesson-8.py`  
**Тема:** оператор `+` для своих объектов

### Теория (5 мин)

- **`__add__`** — что происходит при `a + b`.
- `Point(3, 5) + Point(2, -1)` → новая `Point(5, 4)`.
- Создаётся **новый** объект; старые не меняются.
- `NotImplemented` — если складывать с неподходящим типом.

### Практика (10 мин)

1. Запустите `lesson-8.py`.
2. Посчитайте `start + step + step` — два шага подряд.
3. Попробуйте `Point(0, 0) + Point(10, 0)`.

### Мини-задание (5 мин)

Сложите три точки в цепочке и выведите результат через `print()`.

**Проверка:** умеете складывать точки как векторы через `+`.

## Урок 9. `__mul__` (~20 мин)

**Файл:** `lesson-9.py`  
**Тема:** умножение точки на число

### Теория (5 мин)

- **`__mul__`** — `point * 3` умножает координаты на 3.
- **`__rmul__`** — `3 * point` (число слева); вызывает тот же смысл.
- Удобно для «N шагов за раз» или ускорения.

### Практика (10 мин)

3. Что даёт `Point(2, 1) * 0`?

### Мини-задание (5 мин)

**Проверка:** понимаете разницу между `__add__` (две точки) и `__mul__` (точка × число).

## Урок 10. `@staticmethod` (~20 мин)

**Файл:** `lesson-10.py`  
**Тема:** методы без `self`

### Теория (5 мин)

- Обычный метод: `lamp.turn_on()` — нужен объект.
- **`@staticmethod`**: `Point.origin()` — вызываем от **класса**, объект не нужен.
- Подходит для «фабрик» и проверок: `from_tuple()`, `distance()`, `is_valid_name()`.
- **Константа класса** (`MAX_BRIGHTNESS`) — общая для всех объектов.

### Практика (10 мин)

1. Запустите `lesson-10.py`.
2. Создайте точку через `Point.from_tuple((1, 2))`.
3. Добавьте `@staticmethod mid(a, b)` — середина между двумя точками.

### Мини-задание (5 мин)

Напишите `Lamp.create(name, x, y)`, который сначала проверяет имя через `is_valid_name`.

**Проверка:** знаете, когда метод нужен с `self`, а когда — `@staticmethod`.

## Словарь

| Слово | Пример | Значение |
|-------|--------|----------|
| класс | `class Point` | чертёж |
| объект | `Point(1, 2)` | экземпляр класса |
| атрибут | `point.x` | данные объекта |
| метод | `point.move()` | действие объекта |
| self | первый аргумент метода | ссылка на себя |
| композиция | лампа + Point внутри | «состоит из» |
| `__str__` | `print(point)` | строка для человека |
| `__repr__` | `repr(point)` | строка для отладки |
| `__add__` | `a + b` | своё сложение |
| `__mul__` | `a * 3` | умножение на число |
| `@staticmethod` | `Point.origin()` | метод без объекта |
