# Урок: «Змейка» в pygame (ООП-стиль)

Тот же урок, что в `snake-simple`, но код организован через **классы**. Каждый шаг — отдельный файл.

**Запуск** (из папки `snake-oop`):

```
py snake-1.py
py snake-2.py
...
```

---

## Шаг 1. Класс Game — полотно

**Файл:** `snake-1.py`

**Что делаем:** выносим игру в класс `Game`.

**Главные идеи:**
- `class Game` — «чертёж» игры: окно, цикл, отрисовка
- `__init__` — конструктор: создаёт окно при `Game()`
- `handle_events()` — обрабатывает закрытие окна
- `draw()` — рисует кадр
- `run()` — главный цикл
- `Game().run()` — создали объект и запустили игру

**Проверка:** чёрное окно, закрывается крестиком.

---

## Шаг 2. Класс Snake — голова змейки

**Файл:** `snake-2.py`

**Что добавляем:** класс `Snake` отвечает за змейку.

**Главные идеи:**
- у змейки есть `segments` — список координат частей тела
- метод `draw(screen)` — рисует змейку на экране
- `Game` создаёт `Snake` и вызывает `snake.draw()`

**Проверка:** зелёный квадрат по центру.

---

## Шаг 3. Методы движения и управления

**Файл:** `snake-3.py`

**Что добавляем:** змейка движется сама, стрелки меняют направление.

**Главные идеи:**
- `direction` — направление движения (атрибут объекта)
- `handle_key(key)` — меняет направление по нажатой клавише
- `move()` — делает один шаг вперёд
- `Game` передаёт события змейке: `snake.handle_key(event.key)`

**Проверка:** змейка ползёт, стрелки работают.

---

## Шаг 4. Класс Apple — яблоко

**Файл:** `snake-4.py`

**Что добавляем:** яблоко — отдельный класс.

**Главные идеи:**
- `Apple` хранит позицию `pos` и умеет рисоваться
- `spawn(snake)` — ставит яблоко в случайную клетку, где нет змейки
- каждый объект рисует **себя**: `snake.draw(screen)`, `apple.draw(screen)`

**Проверка:** змейка движется, на поле красное яблоко.

---

## Шаг 5. Рост, столкновения, конец игры

**Файл:** `snake-5.py`

**Что добавляем:** полная логика в методах классов.

**Главные идеи:**
- `Snake.move(apple)` — движение + проверка столкновений; возвращает, съела ли яблоко
- `Apple.respawn(snake)` — новое яблоко после съедания
- `Game.game_over` — флаг конца игры
- `Game.draw_game_over()` — текст на экране

**Проверка:** змейка растёт, игра заканчивается при ударе.

---

## Словарь (ООП)

| Слово | Значение |
|-------|----------|
| класс (class) | чертёж для создания объектов |
| объект | конкретный экземпляр класса: `snake = Snake(...)` |
| `__init__` | метод, который вызывается при создании объекта |
| атрибут | переменная объекта: `self.segments` |
| метод | функция внутри класса: `self.draw()` |
| `self` | ссылка на сам объект |

---

## Сравнение с snake-simple

| snake-simple | snake-oop |
|--------------|-----------|
| переменные `snake`, `apple` | объекты `Snake()`, `Apple()` |
| функция `spawn_apple()` | метод `apple.spawn()` |
| код в одном месте | каждый класс отвечает за своё |
